PROJECT OVERVIEW

Beauty boxes from Korea, reimagined.

UnnieBox is a subscription beauty service curating boxes with Korean skincare and cosmetics, shipped internationally. The challenge: design a complete landing page that builds trust fast, communicates product value visually, and guides users through a simple order flow.

I designed the full UI system — from logo and favicon to every button state, modal, and micro-interaction.

Problem and goals

Problem

Beauty boxes require trust. Users need proof, clarity, and confidence before they share contact details or pay. Generic layouts kill conversions.

Goals

Build a premium look, explain the offer fast, and guide users through a simple flow — learn, trust, order. Every section should answer an objection.

Process

I worked from story structure to reusable components. Each step focused on clarity, hierarchy, and conversion.

01
Story and layout
Defined the page narrative, section order, and CTA rhythm for a smooth user journey.
02
Design system
Set colors, typography, spacing, and card styles to keep everything consistent.
03
Components
Built a UI kit of buttons, inputs, tags, icons, and card templates to speed up design.
04
Interaction and polish
Designed modal flows, micro-interactions, and final visual refinements for a premium feel.

KEY LEARNINGS

What I learned building this.

01
Conversion flows need storytelling
Each section must answer an objection and guide the user to the next step confidently.
02
Design systems save time
A strong UI kit makes every new section faster and keeps the page visually consistent.
03
Typography sets the tone
Editorial display type feels premium but must be balanced with clean UI text.
04
Trust signals must be authentic
Real credentials, clear reviews, and transparent steps build confidence quickly.
