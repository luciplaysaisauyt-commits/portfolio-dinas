Mafia Platform
Project Overview

CLM — the competitive Mafia hub.

CLM is a real platform for tracking player ratings, running tournaments, managing clubs, and recording game sessions for the board game Mafia. The platform had been running for years on an outdated interface — functional but visually overwhelming and painful to use.

This was a real paid client project. My goal: redesign the entire product, simplify navigation, establish a strong dark + red visual identity, and build a scalable UI component system.

Problem and goals

The Problem

The old MafiaRatings site had a cluttered Windows-XP-era layout — dense tables, outdated fonts, no visual hierarchy, complex navigation. New users were lost immediately.

The Goals

Redesign for clarity, speed, and confidence. Create a dark, bold brand identity with red accents. Build a reusable component library that scales with new features.

Design process

I followed a structured approach — starting from user flows and information architecture before touching any visual design.

01
Research & audit
Analyzed the old platform, identified pain points, mapped all user types: players, club owners, admins, and judges.
02
Information architecture
Restructured navigation and page hierarchy. Defined 7 main sections: Home, Clubs, Tournaments, Ratings, Games, Hall of Fame, Info.
03
Design system
Built color palette, typography scale, and a full component library — cards, tables, badges, buttons, forms.
04
Screen design
Designed all 14+ screens with role-based differences: player vs club owner vs admin.

Key Learnings

What I learned.

01
Role-based design is complex
Each user type sees and does different things. Good IA planning before visual work is non-negotiable.
02
Data-heavy UIs need hierarchy
The design job is making data readable at a glance — not just making it look good.
03
Brand identity drives trust
A consistent dark + red visual language makes the platform feel professional and competitive.
04
Systems scale, one-offs don't
A proper component library from the start saved enormous time at screen 10, 12, and 14.
